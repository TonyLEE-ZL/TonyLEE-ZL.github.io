返回类型和return语句

  1. 不要返回局部对象的引用或者指针。一旦函数完成，局部对象被释放，引用的内容不存在，指针也将指向不存在的对象。

  2. 列表初始化返回值：函数可以返回花括号包围的值的列表。在这种情况下，如果函数返回是内置类型，花括号中最多含有一个值，如果函数返回的是类类型，由类本身定义初始值如何使用。

返回数组指针

因为数组不能被拷贝，所以函数不能返回数组，不过却可以返回数组的指针或引用

// 不用typedef

int (*func(int i))[10] // 表示函数返回类型为一个指针，指针指向的是包含10个整数的数组。
  • 使用尾置返回类型:在形参列表后面加上->表示函数的返回值类型，在本应该出现返回类型的地方放置一个auto。

auto fun(int i) -> int(*)[10];

  • 如前文所说，顶层const不影响传入函数的对象

fucn2(int *);

  • 另外一方面，如果形参是某种类型的指针或者引用，则通过区分其指向的是常量对象还是非常量对象实现函数重载，此时的const是底层的。 由于const对象不能转化成非常量对象因此它只能使用const声明的函数，而非常量可以转化成常量，形参列表变量类型相同的所有函数都可以调用，但是编译器会优先选择非常量版本。

重载与作用域

  • 不要将函数的声明置于局部作用域内，这样会屏蔽掉外面的同名重载函数。

特殊用途的语言特性

默认形参

  • 若有多个默认实参，函数调用的时候按照其位置进行解析，默认实参填补函数调用缺少的尾部的实参。

  • 默认实参的声明，多次声明一个函数是合法的，但是同一默认形参只能声明一次，后续的声明只能为之前没有默认值的形参添加默认形参。

constexpr 函数

  • 指能用于常量表达式的函数，遵循以下规定:函数返回类型及所有形参类型都得是字面值类型，而且函数体中必须有且只有一个return语句。 return语句不一定返回常量表达式。

  • 内联函数和constexpr函数可以在程序中多次定义，内联函数和constexpr函数通常定义在头文件中。

函数指针

  1. 当我们把函数名作为一个值使用的时候，该函数自动转换成指针

pf = length

pf = &length //这两种赋值语句都可取

  1. 直接使用指向函数的指针调用该函数，无须提前解引用指针。

    1
2
3

    bool b1 = pf("HELLO", "Good");
bool b2 = (*pf)("HELLO", "Good");
bool b3 = length("HELLO", "Good");

  2. 指向不同类型的指针不存在转换关系，可以为函数指针赋一个0表示没有指向任何一个函数。

函数指针形参

  • 当函数的参数列表中有函数当成形参的时候，会自动转换成函数指针的形式。由于直接使用函数指针显得很冗长，使用类型别名的方法。

typedef decltype (length) *FuncP2

返回指向函数的指针

  • 与函数指针做形参的情况不同，返回类型不会自动转换成指针，必须显式地将返回类型指定为指针。
