变量初始化

  • 未被初始化的的变量被访问和拷贝会引发错误。

  • 定义在函数体外部的变量会被初始化为0。 类对象未初始化时则有其初值决定。定义在函数体内部变量不被初始化。

  • 在变量名之前添加extern 关键字表示声明该变量。若包含初始值就变成定义了。在函数体内部如果想初始化该变量将会引发错误。

const 关键字

  2. 常量必须由指向常量的指针指定 const int */& 可以指定任意变量以及常量

处理变量类型

  1. auto 给分析表达式所属类型 其定义变量必须由初始值

  2. decltype 从表达式中推断要定义变量类型 如果变量的后面由双层括号时返回的类型值总是引用。
